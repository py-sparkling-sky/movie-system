# 观影系统 - 快速开始指南

## 一、项目概述
完整的观影管理系统，包含电影浏览、用户管理、收藏和观影记录功能。

## 二、技术栈
- **前端**: HTML + Bootstrap + JavaScript
- **后端**: Node.js + Express + MySQL
- **认证**: JWT Token

## 三、快速启动

### 1. 创建数据库
```sql
CREATE DATABASE movie_watching_system CHARACTER SET utf8mb4;
```

### 2. 启动后端
```bash
cd backend
# 修改 .env 文件中的数据库密码
npm run dev
```
后端地址: http://localhost:3000

### 3. 访问前端
- 登录页: `frontend/login.html`
- 注册页: `frontend/register.html`
- 首页: `观影.html`

### 4. 默认管理员账号
- 用户名: admin
- 密码: admin123456

## 四、核心功能
1. 用户注册/登录
2. 电影浏览/搜索/筛选
3. 电影收藏
4. 观影记录
5. 后台管理(管理员)

## 五、API接口
- `POST /api/users/register` - 注册
- `POST /api/users/login` - 登录
- `GET /api/movies` - 电影列表
- `GET /api/movies/:id` - 电影详情
- `POST /api/watch/favorites` - 添加收藏
- `GET /api/watch/favorites` - 收藏列表
